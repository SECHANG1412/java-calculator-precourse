# 문자열 덧셈 계산기

## 요구 사항
1. 빈 문자열을 입력하면 0을 반환한다.
2. 쉼표(,) 또는 콜론(:)을 구분자로 사용하여 숫자를 더한다.
3. 커스텀 구분자를 지정할 수 있다. 구분자는 "//"와 "\n" 사이에 위치한 문자를 사용한다.
4. 음수를 입력하면 `IllegalArgumentException`을 던진다.

## 실행 예시
1. 입력: "1,2:3" => 결과: 6
2. 입력: "//;\n1;2" => 결과: 3
3. 입력: "" => 결과: 0
4. 입력: "-1,2" => 오류 발생: 음수는 입력할 수 없습니다: -1
